# Frontend - Personal Trainer

Angular 19 frontend application for the Personal Trainer platform.

## Technologies

- Angular 19.2
- TypeScript 5.7
- SCSS
- Angular Material

## Getting Started

### Prerequisites

- Node.js v22.14.0 or higher
- Backend API running at `http://localhost:5000`

### Installation & Run

```bash
npm install
npm start
```

Application: `http://localhost:4200`

### Build

```bash
npm run build
```

### Tests

```bash
npm test
```

## Project Structure

frontend/
