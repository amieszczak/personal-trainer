# PersonalTrainer

A modern, interactive personal training application built with Angular 19.

Check out the latest build [here](https://amieszczak.github.io/personal-trainer/)

## Technologies

- Angular 19.2
- TypeScript 5.7
- SCSS

## Getting Started

### Prerequisites

- Node.js v22.14.0 or higher
- npm

### Installation

```bash
npm install
```

### Development Server

```bash
npm run start
```

Navigate to `http://localhost:4200/`

### Build

```bash
npm run build
```

Build artifacts will be stored in the `dist/` directory.
